# OctoAcme — Responsibilities Checklist

## Purpose
This comprehensive checklist ensures all critical project activities are assigned and tracked across different project phases. Use this to identify gaps in accountability and ensure nothing falls through the cracks.

## How to Use
1. Review each phase of the project lifecycle
2. Assign owner for each responsibility
3. Track completion status
4. Update regularly during project execution
5. Use as a reference during planning and retrospectives

## Using This Checklist

### Tips for Success:
- Review this checklist during project initiation to ensure all roles are assigned
- Use during weekly syncs to track progress
- Customize based on project size and complexity
- Not all items may apply to every project—adapt as needed
- Review during retrospectives to identify gaps or improvements

### Ownership Guidelines:
- **Primary Role**: Accountable for ensuring the work is completed
- **Supporting Roles**: Contribute expertise and collaborate
- **Assigned To**: Specific individual responsible
- **Status**: Track completion to maintain visibility
